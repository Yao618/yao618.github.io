React Router - 認識名詞 SPA & 環境設定

2022-08-12 React 前端框架 338 個字
React React-Router

本篇將簡單介紹 SPA 與 React Router 的環境設定。

認識名詞 - SPA 單頁應用程式

前端路由的好處是不必每次都重新刷新頁面，而是從頭到尾只需要一個 index.html
之後在頁面上的互動都不需要往返伺服器，換路徑時只會更改頁面裡不一樣的地方。這表示該頁面不會被重新載入。
使用起來就像是一個應用程式。這樣的網頁程式，我們就稱之為Single Page Application(SPA)。

環境設定

步驟一：安裝 Node.js

在使用 React Router 之前，必須確保電腦中已安裝 Node.js 的環境。
未安裝的話，先前往 Node.js官網 進行下載，
下載完成後 npm 會一起被安裝

備註：安裝完請務必重新開機

步驟二：Create React App

打開終端機，輸入以下程式碼，即可快速創建 React 的開發環境並運行。

npx create-react-app my-app
cd my-app
npm start

程式碼中的 my-app 部分，可更改為自己專案的名稱

步驟三：安裝 React Router

在上一步驟創建的 my-app 專案中，打開終端機輸入以下程式碼即可安裝。

npm install react-router-dom@6

到此為止，環境都已經設定完成了。開始撰寫程式碼吧！

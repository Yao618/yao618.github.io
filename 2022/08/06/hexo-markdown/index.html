Markdown - 快速掌握常用語法

2022-08-06 Hexo 1k 個字
Hexo Markdown

本篇將帶你快速掌握 Markdown 語法，讓你流暢撰寫文章！

Markdown 介紹：

Markdown 是一種輕量級標記式語言，同時它的目標是實現「易讀易寫」，擁有純文字的標記特性。
目前許多網站都廣泛使用 Markdown 來撰寫說明文件，如 GitHub 裡面常見的 README.md，
其中 .md 表示的就是 Markdown，就讓我們來看看常用的語法吧！

標題字：

Markdown 的標題總共分為 6 個層級，依據 HTML 的結構對應為 <h1> ~ <h6>
一共有 1 ~ 6 個 #，代表 H1 ~ H6。

使用方式：

# 標題 1
## 標題 2
### 標題 3
#### 標題 4
##### 標題 5
###### 標題 6

展示效果：

標題 1

標題 2

標題 3

標題 4

標題 5
標題 6

備註：標題1 和 標題2 有底線是因為 Hexo Theme 自帶的 CSS 效果

各式字體效果：

字體樣式使用上的方法很接近，都是使用特定符號在文字前後包起來，語法如下。

斜體：

*your text*

粗體：

**your text**

粗體兼具斜體：

***your text***

刪除線：

~your text~

展示效果：

這是一段展示文字，這是斜體文字，這是粗體文字，粗體兼具斜體，這是具有 刪除線 的文字。

連結：

撰寫文章時若要引導讀者前往某個網站，即可在適當的段落插入連結，方便讀者隨時參考相關的內容。

使用方式：

[連結名稱](https://google.com)

效果展示：

連結名稱

階層式區塊：

此語法通常運用在表現結構或功能關西上，是相當好用的呈現方法。

使用方式：

>第一層的顯示區塊
>>第二層的顯示區塊
>>第二層的顯示區塊
>>>第三層的顯示區塊

效果展示：

第一層的顯示區塊

第二層的顯示區塊
第二層的顯示區塊

第三層的顯示區塊

CheckBox：

CheckBox 常用在確認事情是否完成，中括號若有 x 代表此項目是打勾狀態。

使用方式：

- [x] 這是一個已打勾的項目
- [ ] 這是一個未完成的項目

效果展示：

  • 這是一個已打勾的項目
  • 這是一個未完成的項目

清單列表：

使用方式：

此語法常用於同個項目或主題之下要展示出多個內容時，即可使用列表來表示。
例如：開發 APP 時客戶需要哪些功能、專案有哪些內容需要修改。

* 第一項目 
  * 第一項目的文字內容
  * 第一項目的文字內容
* 第二項目 
  * 第二項目的文字內容
  * 第二項目的文字內容

效果展示：

  • 第一項目
    • 第一項目的文字內容
    • 第一項目的文字內容
  • 第二項目
    • 第二項目的文字內容
    • 第二項目的文字內容

程式碼的顯示效果：

許多開發者在撰寫文章時，經常需要把程式碼展示出來，
Markdown 正好提供了我們能夠輸出具有 Highlight 效果的語法。

單行程式碼：

如果要標記一小段行內程式碼，只要在文字前後用反引號把它包起來（`）即可。

使用方式：

javascript 中的 `alert("hello world");` 可顯示提示視窗。

展示效果：

javascript 中的 alert("hello world"); 可顯示提示視窗。

區塊程式碼：

三個連續的反引號（`）用在開頭和結尾，即可展示區塊程式碼。
在反引號後面加上程式語言的名稱，可擁有對應程式語言的 Highlight 效果

使用方式：

```javascript
function multiply(num1,num2) {
  let result = num1 * num2;
  return result;
}
```

展示效果：

function multiply(num1,num2) {
  let result = num1 * num2;
  return result;
}

消息提示：

寫作時需要若需要添加補充或是警告消息，可在 Markdown 中如下書寫，
並將其中的 style 修改為對應的名稱即可。

使用方法：

{% note style %}
輸入內容...
{% endnote %}

展示效果：

這是未更改 style 的樣式

這是 primary 提示

這是 success 提示

這是 info 提示

這是 warning 提示

這是 danger 提示
